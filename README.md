# Шаблон Facade (Python)

Цей проєкт демонструє шаблон проєктування **Facade** на Python.

Шаблон Facade надає спрощений інтерфейс до складної підсистеми.
Він приховує внутрішню складність і дозволяє клієнту взаємодіяти
з системою через одну точку входу.

## Структура
- subsystems/ – внутрішні компоненти системи
- facade/ – реалізація фасаду
- main.py – клієнтський код

## Використання
Система домашнього кінотеатру (DVD-програвач, Проєктор, Аудіосистема)
